# Gemini Code Configuration

This repository extends the raibid-labs organization configuration for Gemini.

## Project Configuration

This project is configured to use:
- SPARC methodology for systematic development
- Gemini for agent coordination
- Organization-wide coding standards

## Development Guidelines

1. Always use concurrent operations in single messages
2. Never save working files to root folder
3. Organize files in appropriate subdirectories
4. Follow SPARC methodology phases

## File Organization

- `/src` - Source code files
- `/tests` - Test files
- `/docs` - Documentation
- `/config` - Configuration files
- `/scripts` - Utility scripts
- `/examples` - Example code
